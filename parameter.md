# Ricoh GR IIIx 影像控制與參數邏輯全表 (V2.0)

本文件定義了 Ricoh GR IIIx (含 HDF 版本) 的影像生成邏輯，旨在將感性的「美學描述」轉化為精確的「相機參數」。

## 一、 全局拍攝設定 (Global Settings)

在進入濾鏡微調前，這三項參數決定了畫面的基調與光影質感。

| 參數名稱 | 建議範圍 / 選項 | 調整邏輯 (Vibe to Logic) |
| :--- | :--- | :--- |
| **HDF 效果** | **ON / OFF** | **ON：** 產生擴散光暈，適合夢幻、溫柔或復古場景。<br>**OFF：** 追求極致銳度與純淨度時使用。 |
| **建議曝光 (EV)** | **-0.3 至 -0.7 EV** | **低曝光：** Ricoh 經典風格，保護高光，使色彩更濃郁深沉。<br>**0 EV：** 適合光比均勻或需要高亮感的日系風格。 |
| **白平衡 (WB)** | **日光 (5200K) / 多功能 AWB** | **日光：** 保持穩定的溫暖調性，適合模擬底片感。<br>**AWB：** 適合光影變化劇烈的街拍，保持色彩準確。 |

## 二、 影像控制基底模式 (Base Image Control)

決定色彩的根本調性。

| 中文名稱 | 英文名稱 | 核心視覺特徵 | 建議應用場景 |

| **單色** | **Monotone** | 階調平滑的標準黑白 | 人文攝影、簡約構圖 |
| **硬單色** | **Hard Monotone** | 邊緣銳利、反差強烈 | 建築線條、硬派紀實 |
| **高對比黑白** | **Hi-Contrast B&W** | 極高反差、粗粒子感 | 森山大道風格、街頭衝擊 |
| **底片** | **Negative Film** | 色彩偏淡、高光帶青調 | 文青感、日常街拍、復古 |
| **正片** | **Positive Film** | 濃郁、飽滿、經典 Ricoh 藍 | 旅遊紀錄、高質感商業感 |
| **跳漂白** | **Bleach Bypass** | 低飽和、高對比、冷硬質感 | 工業風、電影感、城市廢墟 |
| **懷舊** | **Retro** | 泛黃、褪色、低頻對比 | 70年代回憶、陳舊感 |
| **正負逆沖** | **Cross Process** | 強烈偏色 (紫/黃/綠) | 實驗攝影、前衛風格 |

## 三、 詳細微調參數邏輯 (Fine-tuning)

| 參數名稱 (中/英文) | 範圍 / 選項 | 調整邏輯與氛圍引導 (Vibe to Logic) |
| :--- | :--- | :--- |
| **飽和度 (Saturation)** | -4 至 +4 | **增加：** 強化色彩表現，適合鮮豔、活潑、普普藝術風格。<br>**減少：** 降低色彩干擾，適合憂鬱、冷峻、紀實底片感。 |
| **色相 (Hue)** | -4 至 +4 | **正值：** 偏向黃綠色調，模擬復古底片或暖陽感。<br>**負值：** 偏向紅紫色調，適合城市霓虹或特殊人像風格。 |
| **影調 (High/Low Key)** | -4 至 +4 | **正值：** 增加整體亮度，打造日系高調 (High-key) 或空氣感。<br>**負值：** 壓低影調，適合低調 (Low-key)、神祕或硬派氛圍。 |
| **對比度 (Contrast)** | -4 至 +4 | **增加：** 強化明暗反差，增加畫面張力與現代感。<br>**減少：** 讓影調過渡平滑，增加溫柔、復古或平淡的氛圍。 |
| **對比度 - 亮部 (Contrast - Highlight)** | -4 至 +4 | **增加：** 壓抑高光，增加雲朵或光影細節。<br>**減少：** 讓高光溢出，配合 HDF 版本可產生強烈的光暈特效。 |
| **對比度 - 暗部 (Contrast - Shadow)** | -4 至 +4 | **增加：** 讓暗部變淺（洗白），模擬過期底片或低對比黑白。<br>**減少：** 讓暗部更深邃，增加畫面的重量感與厚實度。 |
| **銳度 (Sharpness)** | -4 至 +4 | **增加：** 強化邊緣細節，適合建築、風景或強調材質感的拍攝。<br>**減少：** 柔化邊緣，模擬老鏡頭或追求自然、不數位化的質感。 |
| **色調 (Color Tone)** | 依基底模式而定 | 在單色模式中決定濾鏡色（如金褐色、藍色）；在正負片模式中微調整體冷暖傾向。 |
| **明暗 / 周邊失光 (Shading)** | -4 至 +4 | **正值：** 修正鏡頭失光，使畫面亮度均勻。<br>**負值：** 增加人為暗角，將觀者視線引導至中心，增加電影感。 |
| **清晰 (Clarity)** | -4 至 +4 | **增加：** 強化物體輪廓與微對比，適合硬朗風格。<br>**減少：** 讓畫面呈現朦朧、柔焦感，適合人像或唯美氛圍。 |

## 四、 HDF 特殊連動邏輯 (HDF Optimization)

針對 **Ricoh GR IIIx HDF** 版本，AI 應根據描述自動切換：

1. **極致光暈 (Maximum Glow)：**
   - **配置：** `HDF: ON` + `建議曝光: -0.3 EV` + `對比度(亮部): -3` + `清晰: -2`。
2. **硬派寫實 (Sharp Reality)：**
   - **配置：** `HDF: OFF` + `建議曝光: -0.7 EV` + `銳度: +2` + `清晰: +2`。
